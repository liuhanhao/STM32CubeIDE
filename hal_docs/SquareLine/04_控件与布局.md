# SquareLine Studio — 控件与布局

## 一、添加控件

有两种方式添加控件：

**方式一：** 左侧 Hierarchy 面板底部的 **+ Widget** 按钮
**方式二：** 在画布空白处右键 → **Add Widget**

## 三、控件基本操作

### 3.1 选中与移动

- 单击选中，拖动移动
- 选中后在检查器 **Properties** 面板精确输入 X/Y/W/H

### 3.2 位置与尺寸

检查器 → **Properties** Tab：

| 属性 | 说明 |
|------|------|
| X / Y | 相对于父容器的位置 |
| W / H | 宽度和高度（可输入像素值或百分比）|
| Align | 对齐方式（居中、左上、右下等）|
| X Offset / Y Offset | 对齐后的偏移量 |

**对齐快捷操作：**
选中多个控件后，顶部工具栏会出现对齐按钮（左对齐、居中、等间距等）。

### 3.3 父子关系

控件可以嵌套，子控件坐标相对于父容器：
- 在层级面板拖动控件到另一个控件下方，建立父子关系
- 父容器被删除时，子控件一起删除

## 四、布局系统

### 4.1 自由布局（默认）

手动设置每个控件的 X/Y 坐标，适合固定位置的设计。

### 4.2 Flex 布局

类似 CSS Flexbox，子控件自动排列。

在检查器 **Properties** 中选择容器（Panel），开启 **Layout → Flex**：

| 属性 | 说明 |
|------|------|
| Flex Direction | Row（横向）/ Column（纵向）|
| Flex Wrap | Wrap（换行）/ NoWrap |
| Main Place | 主轴方向对齐（Start / Center / End / Space Between / Space Around）|
| Cross Place | 交叉轴对齐（Start / Center / End）|
| Track Place | 多行时的行间距 |

**使用场景：** 导航栏、按钮组、列表项

### 4.3 Grid 布局

类似 CSS Grid，按行列排列子控件。

在容器属性中开启 **Layout → Grid**，设置列宽和行高（支持像素值、FR 比例、Content 自适应）：

```
列定义：[80, 1FR, 80, LV_GRID_TEMPLATE_LAST]
         ↑固定  ↑弹性  ↑固定
行定义：[50, 1FR, LV_GRID_TEMPLATE_LAST]
```

**使用场景：** 表格、图标网格、仪表盘

## 五、控件命名规范

控件名称直接影响导出的 C 变量名：

```
ui_Screen1_Label1   →  lv_obj_t *ui_Screen1_Label1
```

建议按功能命名，导出后的代码更易读：

```
Label_Title         →  ui_Label_Title
Button_Start        →  ui_Button_Start
Panel_Header        →  ui_Panel_Header
Bar_Progress        →  ui_Bar_Progress
```

## 六、复制与复用

### 复制控件

`Cmd + D`：在原位置复制一个相同的控件（包括样式和子控件）。

### 创建组件（Component）

重复使用的控件组合可以保存为组件：
1. 选中控件（可多选）
2. 右键 → **Create Component**
3. 命名后，组件出现在资源面板 **Components** 列表
4. 从列表拖放到画布即可复用

修改组件后，所有使用该组件的地方同步更新。
