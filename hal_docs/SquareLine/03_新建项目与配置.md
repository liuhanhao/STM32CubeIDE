# SquareLine Studio — 新建项目与配置

## 一、新建项目

启动后点击 **Create → New Project**，进入项目创建向导。

---

## 二、项目配置参数

### 2.1 基础设置

| 参数 | 说明 | 常用值 |
|------|------|--------|
| **Project Name** | 项目名称，影响导出的文件夹名 | `my_ui` |
| **Project Path** | 保存位置 | 自定义 |

---

### 2.2 显示设置（重要）

| 参数 | 说明 | 示例 |
|------|------|------|
| **Resolution** | 屏幕分辨率，必须与实际屏幕一致 | 240×320 / 320×480 / 480×800 |
| **Color Depth** | 颜色深度，必须与 `lv_conf.h` 一致 | 16 bit（RGB565，SPI 屏）/ 32 bit |

> ⚠️ 分辨率和颜色深度设置错误是最常见的问题，后期无法修改，新建前务必确认。

**常见屏幕配置参考：**

| 屏幕型号 | 分辨率 | 颜色深度 |
|---------|--------|---------|
| ILI9341 SPI | 240×320 | 16 bit |
| ST7789 SPI | 240×240 / 240×320 | 16 bit |
| ILI9486 | 320×480 | 16 bit |
| 7寸 RGB 屏 | 800×480 | 16 bit |

---

### 2.3 LVGL 版本

| 选项 | 说明 |
|------|------|
| LVGL 8.x | 大多数现有工程使用的版本 |
| LVGL 9.x | 最新版，v1.6.0 起支持 |

> 必须与工程中 `idf_component.yml` 或 `lib_deps` 里声明的 LVGL 版本一致。

---

### 2.4 导出语言

| 选项 | 说明 |
|------|------|
| C | 嵌入式开发（ESP32/STM32）必选 |
| MicroPython | MicroPython 项目选这个 |

---

## 三、项目文件结构

创建后，项目保存为 `.spj` 格式（SquareLine Project）：

```
my_ui/
├── my_ui.spj          ← 项目文件，双击用 SquareLine Studio 打开
├── assets/
│   ├── images/        ← 导入的图片
│   └── fonts/         ← 导入的字体
└── components/        ← 自定义组件
```

---

## 四、修改已有项目的设置

**Project → Project Settings**（或顶部菜单）可修改部分设置：

| 可修改 | 不可修改 |
|--------|---------|
| LVGL 版本 | 分辨率 |
| 导出路径 | 颜色深度 |
| 主题颜色 | — |

> 分辨率和颜色深度设置后不能改，需要重建项目。

## 五、典型项目配置示例

### ESP32 + ILI9341（240×320 SPI 屏）

```
Project Name:   esp32_monitor
Resolution:     240 × 320
Color Depth:    16 bit
LVGL Version:   8.3.x
Language:       C
```

### ESP32-S3 + 7寸 RGB 屏

```
Project Name:   esp32s3_hmi
Resolution:     800 × 480
Color Depth:    16 bit
LVGL Version:   8.3.x
Language:       C
```

### STM32 + ST7789

```
Project Name:   stm32_ui
Resolution:     240 × 240
Color Depth:    16 bit
LVGL Version:   8.3.x
Language:       C
```

## 六、屏幕管理

一个项目可以有多个屏幕（Screen），对应多个 UI 页面。

### 添加屏幕

顶部工具栏点击 **+ Add Screen** 或 **File → Add Screen**。

### 屏幕命名建议

```
Screen_Home      ← 首页
Screen_Setting   ← 设置页
Screen_Detail    ← 详情页
```

命名清晰，导出的 C 文件会对应命名：`ui_Screen_Home.c`。

### 屏幕属性

在层级面板选中屏幕根节点，检查器面板中可设置：
- 背景颜色
- 背景图片
- 进入/退出动画
