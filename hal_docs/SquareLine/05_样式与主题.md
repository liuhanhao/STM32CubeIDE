# SquareLine Studio — 样式与主题

## 一、样式面板

选中控件后，检查器右侧切换到 **Style** Tab，可以设置控件的所有外观属性。

---

## 二、Part（部件）

LVGL 控件由多个部件组成，每个部件可以独立设置样式。

Style 面板顶部的下拉框选择要设置哪个部件：

| Part | 说明 | 适用控件 |
|------|------|---------|
| **Main** | 控件主体（背景、边框）| 所有控件 |
| **Scrollbar** | 滚动条 | 可滚动容器 |
| **Indicator** | 指示器（填充部分）| Bar、Slider、Arc |
| **Knob** | 旋钮（拖动点）| Slider、Arc |
| **Selected** | 选中项 | Roller、Dropdown |
| **Items** | 列表项 | Roller、Dropdown |
| **Cursor** | 光标 | Textarea |

---

## 三、State（状态）

同一个 Part 在不同状态下可以有不同样式，State 下拉框选择：

| State | 说明 |
|-------|------|
| **Default** | 默认状态 |
| **Pressed** | 按下时 |
| **Focused** | 获得焦点时 |
| **Checked** | 选中时（Switch、Checkbox）|
| **Disabled** | 禁用时 |

**示例：** 按钮按下时背景色变深
- Part = Main，State = Default → 背景色 `#2196F3`
- Part = Main，State = Pressed → 背景色 `#1565C0`

## 四、常用样式属性

### 背景

| 属性 | 说明 |
|------|------|
| Background Color | 背景颜色（点击色块打开颜色选择器）|
| Background Opacity | 背景透明度（0~255）|
| Background Gradient | 渐变色（颜色2 + 渐变方向）|
| Background Image | 背景图片（从资源面板选择）|

### 边框

| 属性 | 说明 |
|------|------|
| Border Color | 边框颜色 |
| Border Width | 边框宽度（px）|
| Border Side | 边框显示位置（全部/上/下/左/右）|
| Outline | 外轮廓（在边框外侧）|

### 形状

| 属性 | 说明 |
|------|------|
| Radius | 圆角半径（px，`9999` = 完整圆形）|
| Clip Corner | 是否裁剪超出圆角的子内容 |

### 阴影

| 属性 | 说明 |
|------|------|
| Shadow Width | 阴影扩散宽度 |
| Shadow Color | 阴影颜色 |
| Shadow Opacity | 阴影透明度 |
| Shadow X / Y Offset | 阴影偏移 |

### 内边距

| 属性 | 说明 |
|------|------|
| Padding Top/Bottom/Left/Right | 四方向内边距 |
| Padding Row / Column | Flex/Grid 布局的行/列间距 |

### 文字

| 属性 | 说明 |
|------|------|
| Text Color | 文字颜色 |
| Text Font | 字体选择（内置或已导入的自定义字体）|
| Text Letter Space | 字间距 |
| Text Line Space | 行间距 |
| Text Decor | 装饰（下划线、删除线）|
| Text Align | 对齐（左/中/右）|

## 五、全局颜色（Global Colors）

**Project → Global Colors** 可以定义项目级别的颜色变量，类似设计稿中的色板。

```
Primary:     #2196F3   ← 主色调（蓝色）
Secondary:   #4CAF50   ← 辅助色（绿色）
Background:  #1a1a2e   ← 背景色
Text:        #E0E0E0   ← 文字色
Accent:      #FF5722   ← 强调色
```

定义后，控件样式中可以引用这些颜色变量，修改全局颜色时所有使用该颜色的控件同步更新。

---

## 六、主题（Theme）

**Project → Theme** 可以切换内置主题：

| 主题 | 说明 |
|------|------|
| Default Dark | 深色主题（深色背景，浅色文字）|
| Default Light | 浅色主题 |
| Mono | 单色（适合黑白屏）|

主题设置控件默认样式，自定义的样式会覆盖主题默认值。

---

## 七、样式操作技巧

### 复制样式

右键控件 → **Copy Style**，再到另一个控件右键 → **Paste Style**，快速复用样式。

### 重置样式

在 Style 面板每个属性旁边有 **×** 按钮，点击可重置为主题默认值。

### Local Style vs Global Style

- **Local Style（内联样式）**：直接设置在控件上，只影响这一个控件
- **Global Style（全局样式）**：SquareLine Studio 目前主要通过 Global Colors 和 Theme 实现全局样式，代码层面可配合 LVGL 的 `lv_style_t` 使用

---

## 八、导出的样式代码

SquareLine Studio 的样式会导出为内联样式调用：

```c
/* 导出示例 */
lv_obj_set_style_bg_color(ui_Button_Start,
    lv_color_hex(0x2196F3), LV_PART_MAIN | LV_STATE_DEFAULT);
lv_obj_set_style_bg_color(ui_Button_Start,
    lv_color_hex(0x1565C0), LV_PART_MAIN | LV_STATE_PRESSED);
lv_obj_set_style_radius(ui_Button_Start, 8, LV_PART_MAIN | LV_STATE_DEFAULT);
lv_obj_set_style_text_font(ui_Label_Title,
    &lv_font_montserrat_16, LV_PART_MAIN | LV_STATE_DEFAULT);
```
