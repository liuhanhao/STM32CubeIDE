## 一、产品概述

本模块是基于 **SU-03T** 芯片的低成本、低功耗、小体积离线语音识别开发板，适用于智能家居、小家电、灯具、玩具等需要语音控制的场景。

### 主要特性
- USB 5V 供电（也可通过 VCC 引脚输入 5V）
- **无需编程**，通过网页平台配置，会打字即可操作
- 支持自定义唤醒词、命令词、回复语
- 支持自定义开机播报语音
- 支持自定义 GPIO 口控制
- 支持串口输出（UART）
- 支持多种真人发音人选择
- 可无限次修改配置
- 板载：Micro USB 接口、CH340N USB 转串口芯片、高灵敏度麦克风、8Ω 1W 喇叭、拨动开关

## 二、SU-03T 芯片规格

### 核心参数

| 参数 | 规格 |
|------|------|
| 模块型号 | SU-03T |
| 封装 | SMD18 / DIP18 |
| 尺寸 | 21×15×3 (±0.2) mm |
| 内核 | 32-bit RISC，运行频率 240MHz |
| 内置 FLASH | 2MB |
| IO 口数量 | 8 个 |
| 供电电压 | 3.6V ~ 5.5V（推荐 5V，供电电流 >200mA） |
| 平均工作电流 | 60mA |
| 工作温度 | -20℃ ~ 85℃ |
| IO 电平 | 3.3V |

### 支持接口
- UART（全双工，最高 3Mbps）
- SPI Master / Slave（最高 30MHz）
- I2C 主/从（最高 400kHz）
- I2S input/output
- ADC（12-bit SAR，最大 450kHz 采样率）
- PWM × 2
- GPIO（均可配置为外部中断/唤醒源）

### 内置功能
- DSP 指令集 + FPU 浮点运算单元
- FFT 加速器（最大 1024 点复数 / 2048 点实数）
- 3W 单声道 AB 类功放
- 1 路驻极体麦克风接口
- 5V 转 3.3V LDO（外部负载 ≤ 150mA）
- POR、低电压检测、看门狗

### 喇叭驱动能力
| 负载 | 输出功率 |
|------|---------|
| 4Ω（VCC=5V）| 最高 2.9W |
| 8Ω（VCC=5V）| 最高 1.8W |

## 五、与 STM32 通信（串口）

模块通过 **UART1（B6/B7）** 对外输出识别结果。

### 连接方式（模块 ↔ STM32）
```
SU-03T B7 (UART1_TXD)  →  STM32 USARTx_RX
SU-03T B6 (UART1_RXD)  →  STM32 USARTx_TX
SU-03T GND             →  STM32 GND
SU-03T VCC             →  5V 供电（STM32 板 5V 引脚）
```

> IO 电平为 **3.3V**，与 STM32F103 直接兼容，无需电平转换。

### 串口参数（参考配置）
- 波特率：通常为 **9600** 或 **115200**（以平台配置为准）
- 数据位：8
- 停止位：1
- 校验：无

> 具体串口输出格式（十六进制帧或 ASCII）取决于平台编译的固件配置，建议在平台"离线命令词"页面查看每条命令的串口输出值。

## 六、固件配置与烧录流程

### 6.1 平台网址
- 主平台：[http://www.smartpi.cn](http://www.smartpi.cn/#/)
- 论坛教程：[http://bbs.aimachip.com](http://bbs.aimachip.com)

### 6.2 软件配置流程

1. **注册账号** — 在平台注册
2. **创建产品** — 选择"纯离线方案"，芯片选 **SU-03T**，语言选中文
3. **配置 Pin 脚**
   - 继电器默认接 A25，高电平有效
   - 如需上电记忆（掉电恢复上次状态），开启"上电记忆"开关
4. **自定义唤醒词** — 注意平台规则，避免空格和特殊字符
5. **自定义命令词与回复语** — 可配置 TurnOn / TurnOff 等指令
6. **免唤醒词设置**（可选）— 无需先说唤醒词即可直接控制
7. **发音人配置** — 根据喜好选择真人发音
8. **优化配置**（重点）
   - 阈值越高，灵敏度越高，但误识别率也增加
   - 建议多次调试找到平衡点
9. **生成新版本** → **生成 SDK** → **编译固件**
10. **下载固件** — 解压后取 `jx_su_03t_release_update.bin`

### 6.3 硬件烧录连接（USB 转 TTL）

```
AI 离线语音模块    CH340 USB转串口模块
     5V      ←→       5V
     GND     ←→       GND
     TXD     ←→       RXD
     RXD     ←→       TXD
```

### 6.4 烧录软件操作步骤

1. 解压 SDK 压缩包，进入 `Hummingbird-M-Update-Tool` 文件夹
2. 将编译好的 `jx_su_03t_release_update.bin` 复制到该文件夹
3. 双击运行 `UniOneUpdateTool.exe`
4. 选择固件文件，点击"烧录"
5. **多次拨动开关**（或拔插跳线帽），让设备重启，直到软件识别到端口并开始烧录
6. 等待烧录完成，测试指令

> 需要多次拨动的原因：设备仅在**启动瞬间**检测串口升级信号，波特率识别可能失败，多次重启可提高成功率。

## 八、编程注意事项

1. **IO 电平 3.3V**，与 STM32F103 兼容，无需电平转换
2. **烧录口（B0/B1）与通信口（B6/B7）不同**，编程时使用 UART1（B6/B7）
3. 继电器控制引脚默认 **A25，高电平有效**
4. 串口输出的命令 ID/格式需在云平台配置页面确认（每条命令词可设置不同的串口输出值）
5. 模块供电需 **>200mA**，STM32 的 3.3V 引脚无法直接供电，需接 5V
6. 3V3 引脚可为外部小负载供电，但不超过 150mA
7. **使用标准固件时，B6（风扇 PWM）和 B7（灯光 PWM）已被占用**，无法再作为 UART1 使用。若需 STM32 读取串口识别结果，建议使用出厂简单固件，或在平台重新定制固件将串口输出分配到 B2/B3
8. 读取识别结果的两种思路：
   - **串口方式**：使用简单固件或自定义固件，STM32 通过 UART 接收命令 ID
   - **GPIO 电平检测方式**：使用标准固件，STM32 直接检测 A25/A26/A27 等引脚的高低电平变化
