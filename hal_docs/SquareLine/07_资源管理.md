# SquareLine Studio — 资源管理

## 一、资源面板

左侧 **Assets** 面板分两个 Tab：
- **Images** — 项目中导入的图片
- **Fonts** — 项目中添加的字体

---

## 二、图片资源

### 2.1 支持的图片格式

| 格式 | 说明 |
|------|------|
| PNG | 支持透明通道，推荐 |
| JPG | 无透明通道 |
| BMP | 无损，文件较大 |

> SquareLine Studio 内部会自动将图片转换为 LVGL 所需的格式，导出时根据配置输出 C 数组或二进制文件。

### 2.2 导入图片

**方法一：拖放**
直接将图片文件拖入 SquareLine Studio 窗口。

**方法二：面板按钮**
Assets 面板 → Images Tab → **+ 按钮** → 选择图片文件。

导入后图片出现在 Images 列表中。

### 2.3 使用图片

**方式一：作为 Image 控件的图源**
1. 添加 Image 控件到画布
2. 检查器 Properties → **Source** → 从下拉列表选择已导入的图片

**方式二：作为控件背景图**
1. 选中任意控件
2. Style → Main → **Background Image** → 选择图片

**方式三：直接拖放图片到画布**
从 Assets → Images 列表中，将图片拖到画布，自动创建 Image 控件。

### 2.4 切图规范建议

从 Figma / Sketch / Photoshop 导出切图时：

| 建议 | 原因 |
|------|------|
| 使用 PNG 格式 | 支持透明通道，边缘干净 |
| 导出为实际尺寸（1x）| 单片机屏幕不是高 DPI |
| 图片宽高为偶数 | LVGL RGB565 格式对齐要求 |
| 背景透明的图标用 PNG | JPG 会有白色背景 |
| 图片尺寸不要超过屏幕 | 占用 Flash/RAM |

### 2.5 图片导出格式设置

**Project Settings → Image Export Format**：

| 选项 | 说明 | 适用场景 |
|------|------|---------|
| **C Array** | 编译进固件，存 Flash | 图片少、固定不变 |
| **Binary** | 导出 `.bin` 文件，放文件系统 | 图片多、需要更新 |

> 使用文件系统（SD 卡 / SPIFFS）时，选 Binary 并配合 LVGL 文件系统驱动使用。
> 详见 [LVGL_FileSystem.md](../LVGL/LVGL_FileSystem.md)

## 三、字体资源

### 3.1 内置字体

SquareLine Studio 内置 Montserrat 字体的多个尺寸，可直接使用：

在控件 Style → Text Font 下拉框中选择 `Montserrat_XX`（XX 为字号）。

**只支持英文和数字**，不包含中文。

### 3.2 添加自定义字体（支持中文）

**步骤一：准备字体文件**

用命令行工具 `lv_font_conv` 将 TTF 转换为 LVGL C 数组：

```bash
# 安装工具
npm install -g lv_font_conv

# 生成中文字体（16px，常用汉字 + ASCII）
lv_font_conv \
  --font SourceHanSansCN-Regular.ttf \
  --size 16 \
  --bpp 4 \
  --range 0x20-0x7E \
  --range 0x4E00-0x9FFF \
  --format lvgl \
  -o lv_font_cn_16.c
```

或使用在线工具：https://lvgl.io/tools/fontconverter

**步骤二：导入到 SquareLine Studio**

Assets 面板 → Fonts Tab → **+ 按钮** → 选择生成的 `.c` 字体文件。

导入后字体出现在 Fonts 列表中。

**步骤三：在控件中使用**

Style → Text Font → 下拉框中选择刚导入的字体名。

### 3.3 字体导出

导出 UI 文件时，SquareLine 会将字体 `.c` 文件复制到导出目录的 `fonts/` 文件夹。

## 四、资源文件大小参考

| 资源类型 | 大约占用 Flash |
|---------|--------------|
| 小图标 48×48 px（PNG with alpha）| ~9KB |
| 背景图 240×320 px（RGB565）| ~150KB |
| ASCII 字体 16px 4bpp | ~10KB |
| 常用汉字 2500 字 16px 4bpp | ~200KB |
| 全部汉字 20902 字 16px 4bpp | ~1.5MB |

> ESP32-S3（N16R8）有 16MB Flash，资源充裕。
> 普通 ESP32（4MB Flash）需要精简字体范围，或改用 SPIFFS/SD 卡存储资源。

## 六、从 Figma 导出切图的工作流

1. **Figma** 中完成 UI 设计
2. 选中需要导出的图层/组 → 右侧 Export → 格式选 **PNG**，倍率选 **1x**
3. 将导出的 PNG 文件导入 SquareLine Studio
4. 重建 SquareLine 中的布局，图片用于背景或 Image 控件
5. 非图片部分（标签、按钮）在 SquareLine 中用控件实现

> 实际工作中，设计稿中的背景、图标用切图，交互控件（按钮、滑块）用 LVGL 原生控件，
> 这样动效和交互逻辑更灵活。
