## 二、各面板说明

### 顶部菜单栏

| 菜单 | 功能 |
|------|------|
| File | 新建、打开、保存项目 |
| Edit | 撤销、重做、复制、粘贴 |
| View | 显示/隐藏各面板、网格、标尺 |
| Export | 导出 UI 文件 |
| Project | 项目设置（分辨率、颜色深度、LVGL版本）|

顶部工具栏还包含：
- **屏幕选择下拉框** — 切换当前编辑的屏幕
- **Add Screen** — 新增屏幕
- **Run** — 在软件内预览运行效果
- **Export UI Files** — 快速导出按钮

---

### 画布（Canvas）

中央编辑区域，所有控件拖放和布局操作都在这里完成。

**画布操作：**

| 操作 | 方式 |
|------|------|
| 平移画布 | 按住空格 + 拖动 / 中键拖动 |
| 缩放画布 | Ctrl + 滚轮 / 触控板双指缩放 |
| 选中控件 | 单击 |
| 多选控件 | Shift + 单击 / 框选 |
| 移动控件 | 拖动 / 方向键（1px）/ Shift + 方向键（10px）|
| 调整尺寸 | 拖动控件边缘/角落的控制点 |

---

### 层级面板（Hierarchy）

左侧面板，显示当前屏幕的控件树结构，类似 HTML 的 DOM 树。

```
Screen1
├── Panel_Header
│   ├── Label_Title
│   └── Img_Logo
├── Panel_Content
│   ├── Button_Start
│   └── Label_Status
└── Panel_Footer
```

**操作：**
- 单击选中控件
- 拖动调整父子关系
- 右键菜单：复制、删除、重命名、创建组件

---

### 检查器面板（Inspector）

右侧面板，显示选中控件的所有属性，分为几个 Tab：

| Tab | 内容 |
|-----|------|
| **Properties** | 位置（X/Y）、尺寸（W/H）、对齐方式 |
| **Style** | 背景色、边框、圆角、字体、内边距、阴影 |
| **Flags** | Hidden、Clickable、Scrollable 等标志 |
| **Events** | 绑定事件和动作 |

---

### 资源面板（Assets）

左下角面板，管理项目中的图片和字体资源。

- **Images** — 已导入的图片列表，可拖放到画布
- **Fonts** — 已添加的字体，可设为控件的文字字体

---

## 三、常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd + Z` | 撤销 |
| `Cmd + Shift + Z` | 重做 |
| `Cmd + C` | 复制控件 |
| `Cmd + V` | 粘贴控件 |
| `Cmd + D` | 复制并就地粘贴（Duplicate）|
| `Delete` | 删除选中控件 |
| `Cmd + A` | 全选当前屏幕的控件 |
| `Cmd + G` | 将选中控件打组（创建 Panel）|
| `Cmd + +` / `Cmd + -` | 画布缩放 |
| `Cmd + 0` | 画布适配窗口 |
| `F5` | 预览运行 |
| `Cmd + E` | 导出 UI 文件 |

> Windows/Linux 将 `Cmd` 替换为 `Ctrl`

---

## 四、预览功能

点击顶部 **▶ Run** 按钮，在软件内实时预览 UI：

- 可以点击按钮、触发事件、测试屏幕跳转
- 模拟触摸输入（鼠标点击）
- 不需要烧录到硬件即可验证交互逻辑

---

## 五、面板布局自定义

所有面板都可以拖动重新排列，也可以通过 **View → Panels** 显示/隐藏。
关闭软件时会保存面板布局，下次打开恢复。
