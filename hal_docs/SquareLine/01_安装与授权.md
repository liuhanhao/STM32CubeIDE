# SquareLine Studio — 安装与授权

## 一、系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10+（64-bit）/ macOS 11+ / Ubuntu 20.04+ |
| 内存 | 4GB RAM（推荐 8GB+） |
| 显卡 | 支持 OpenGL 3.3+ |
| 磁盘 | 安装约 500MB |

---

## 二、下载

官方下载页面：https://squareline.io/downloads

页面提供三个平台的安装包：

| 平台 | 文件格式 |
|------|---------|
| macOS | `.dmg` |
| Windows | `.exe` 安装程序 |
| Linux | `.AppImage` |

下载对应平台版本，当前最新版为 **v1.6.x**。

---

## 三、macOS 安装步骤

1. 下载 `.dmg` 文件
2. 双击打开，将 `SquareLine Studio.app` 拖入 `Applications` 文件夹
3. 首次启动时，macOS 可能提示"无法验证开发者"：
   - 打开 **系统设置 → 隐私与安全性**
   - 找到 SquareLine Studio 提示，点击 **仍要打开**
4. 启动后登录账号或以 Guest 模式使用

---

## 四、授权说明

SquareLine Studio 提供三种授权级别：

### Personal（免费）

- 完全免费，无时间限制
- **限制：**
  - 最多 5 个屏幕（Screen）
  - 最多 200 个组件
  - 项目名称会带水印标识（不影响导出代码）
- 适合：个人学习、原型验证、小型项目

### Professional（付费，单人）

- 无屏幕和组件数量限制
- 优先技术支持
- 适合：独立开发者、商业产品

### Enterprise（付费，团队）

- 团队多人授权
- 优先支持 + 定制功能

> **个人学习和嵌入式项目开发，Personal 版完全够用。**

---

## 五、账号注册与登录

1. 访问 https://squareline.io 注册账号
2. 启动 SquareLine Studio 后点击右上角 **Sign In**
3. 登录后 Personal 版功能自动激活
4. 不登录也可以以 **Guest** 模式使用，功能与 Personal 相同但无法保存到云端

---

## 六、版本说明

| 版本 | LVGL 支持 | 说明 |
|------|----------|------|
| v1.6.x（当前最新） | LVGL 8.x + 9.3 | 推荐使用 |
| v1.5.x | LVGL 8.x | 稳定版 |
| v1.4.x | LVGL 8.x | 旧版 |

> 导出代码时需要与工程中使用的 LVGL 版本对应，否则 API 可能不兼容。
